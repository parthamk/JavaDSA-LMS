## üì¶ Installation

### Prerequisites

- Node.js >= 16.0.0
- npm or yarn

### Quick Start

1. **Clone the Repository**

   ```bash
   git clone https://github.com/parthamk/JavaDSA-LMS.git
   cd JavaDSA-LMS
   ```
2. **Backend Setup**

   ```bash
   cd backend
   npm install
   cp .env.example .env
   npm run dev
   ```
3. **Frontend Setup** (in a new terminal)

   ```bash
   cd frontend
   npm install
   cp .env.example .env
   npm run dev
   ```

Visit `http://localhost:5173` in your browser.

For detailed setup instructions, see [SETUP.md](./SETUP.md).
